# Telegram Portfolio Bot 🚀
Профессиональный бот-визитка для сбора заявок и управления контентом.

## Стек технологий:
- **Python 3.12**
- **Aiogram 3.x** (Асинхронность)
- **SQLAlchemy** (ORM для работы с БД)
- **SQLite** (Надежное хранение данных)

## Основные функции:
- ⚡ **SPA-интерфейс:** плавное переключение меню без лишнего мусора в чате.
- 📋 **FSM Анкета:** пошаговый сбор данных клиента.
- 🛠 **Админ-панель:** статистика юзеров и массовая рассылка.
